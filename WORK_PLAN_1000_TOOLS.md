## 🏗️ 工具分析模板

### 每个工具必须包含

```json
{
  "name": "工具名称",
  "category": "分类",
  "url": "官网链接",
  "logo": "favicon链接",
  "description": "一句话描述",
  "heat_score": 85,
  "price": {
    "type": "free|paid|freemium",
    "details": "具体价格说明"
  },
  "pros": [
    "优点1：具体说明",
    "优点2：具体说明",
    "优点3：具体说明"
  ],
  "cons": [
    "缺点1：具体说明",
    "缺点2：具体说明"
  ],
  "best_for": ["适用场景1", "适用场景2"],
  "alternatives": ["替代工具1", "替代工具2"]
}
```

### 分析标准

| 维度 | 要求 |
|------|------|
| **优点** | 必须实际测试或调研，不能复制官网 |
| **缺点** | 必须客观指出，不能全是好评 |
| **热度** | 根据真实用户评价、市场份额评分 |
| **价格** | 详细列出免费额度和付费方案 |
| **对比** | 提供2-3个同类替代工具 |

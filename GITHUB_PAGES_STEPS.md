# GitHub Pages 启用步骤 - 图文指引

## 第一步：进入设置

在仓库页面，点击顶部菜单栏的 **Settings**

```
仓库首页
├─ <> Code          
├─ 🔃 Issues        
├─ 📝 Pull requests  
├─ 🚀 Actions       
├─ 📊 Projects      
├─ 📚 Wiki          
├─ 🔒 Security      
├─ 📈 Insights      
└─ ⚙️ Settings  ← 点击这里
```

---

## 第二步：找到Pages选项

进入 Settings 页面后，看**左侧边栏**：

```
Settings
├─ General          ← 默认在这里
├─ Access
├─ Code and automation
│   ├─ Branches
│   ├─ Tags
│   └─ Actions
├─ Security
├─ Code security and analysis
├─ Deploy keys
├─ Webhooks
├─ Notifications
├─ Pages            ← 点击这里！👈
└─ Moderation options
```

**点击左侧的 "Pages"**

---

## 第三步：配置Pages

进入 Pages 页面后，你会看到：

```
GitHub Pages
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Build and deployment

Source
○ Deploy from a branch      ← 选择这个（点击圆圈）
○ GitHub Actions

Branch
┌─────────────────────────────┐
│ main  /  (root)    ▼        │  ← 确保选择 main
└─────────────────────────────┘
                              [Save]  ← 点击保存按钮

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

---

## 第四步：完成！

点击 Save 后，页面上方会显示：

```
✅ Your site is ready to be published at
   https://Sangil1127.github.io/ai-tools-nav/
```

等待 2-5 分钟后，点击这个链接即可访问！

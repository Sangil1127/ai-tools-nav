# AI工具导航 - 最简单的预览方法

## 方法1：直接双击打开（最简单！）

**在Windows上：**
1. 找到文件 `index.html`
2. **双击打开** → 自动用浏览器打开

**在手机上：**
1. 用微信/QQ发送 `mobile.html` 给自己
2. 点击文件 → 选择浏览器打开

---

## 方法2：在线预览（无需安装）

### 使用 CodePen 预览
1. 访问 https://codepen.io/pen/
2. 把 `index.html` 的内容复制到 HTML 框
3. 自动实时预览！

### 使用 JSFiddle 预览
1. 访问 https://jsfiddle.net/
2. 把 HTML 粘贴到 HTML 面板
3. 点击 Run

---

## 方法3：VS Code 预览（推荐开发者）

1. 安装 VS Code
2. 安装扩展 **"Live Server"**
3. 右键点击 `index.html` → **"Open with Live Server"**
4. 自动打开浏览器并实时刷新

---

## 方法4：Python一键启动

**Windows:**
```cmd
cd 网站目录
python -m http.server 8080
```

**然后访问:** http://localhost:8080/

---

## 手机查看效果

用手机扫描或访问：
```
https://your-username.github.io/ai-tools-nav/mobile.html
```

或者直接发送 `mobile.html` 到手机，用浏览器打开。

## 方法1：左侧菜单找

**看页面左侧边栏：**

```
概览        ← 你现在在这里
效果
网址检查
索引
├── 网页
└── Sitemaps   ← 点这个！
体验

安全与手动操作
...
```

**点击 "索引" → "Sitemaps"**

## 方法2：直接访问链接

**复制这个链接到浏览器打开：**


## 方法3：从顶部菜单找

**看页面顶部：**

```
概览 | 效果 | 网址检查 | 索引 | 体验 | 增强功能 | ...
```

**点击 "索引"**，然后在下拉菜单中点击 **"Sitemaps"**

## 📋 操作步骤

1. **找到并点击 "Sitemaps"**
2. **在输入框填：** `sitemap.xml`
3. **点击 "提交"**
4. **完成！** ✅

## 🚀 立即部署

### 步骤1：配置GitHub仓库

**执行以下命令（替换YOUR_USERNAME）：**

```bash
cd /root/.openclaw/workspace/projects/ai-tools-nav

# 添加远程仓库（替换YOUR_USERNAME为你的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/ai-tools-nav.git

# 重命名分支为main
git branch -m master main

# 添加所有文件
git add .

# 提交
git commit -m "v1.0: AI工具导航上线 - 收录10个AI写作工具，SEO优化"

# 推送到GitHub
git push -u origin main
```

### 步骤2：启用GitHub Pages

1. 访问 `https://github.com/YOUR_USERNAME/ai-tools-nav/settings/pages`
2. **Source**: Deploy from a branch
3. **Branch**: main / (root)
4. 点击 **Save**

### 步骤3：访问网站

等待2-5分钟后访问：
```
https://YOUR_USERNAME.github.io/ai-tools-nav/
```
